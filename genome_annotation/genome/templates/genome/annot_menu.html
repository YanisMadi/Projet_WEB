Bienvenue dans l'application CYPS Genome Annotation !


Cette application web a été créée par quatre étudiants en Biologie Computationnelle (AMI2B) à l'Université Paris-Saclay. CYPS genome annotation regorge d'informations sur les génomes, les gènes et les protéines bactériens.
T"Le but de ce site web est de faire des annotations fonctionnelles sur les gènes et les protéines. Les nouvelles annotations seront validées ou rejetées avant d'être soumises à la base de données. La base de données est
écrite en SQLite. En tant qu'utilisateur, vous avez le droit de choisir votre rôle en tant que lecteur, validateur ou annotateur :

- Si vous vous inscrivez en tant que lecteur, vous ne pouvez qu'effectuer des recherches, lire des informations sur les génomes bactériens, les gènes et les protéines. Vous pouvez également vérifier les annotations en cours de validation et télécharger les données de la base de données.

- Cependant, si vous vous inscrivez en tant qu'annotateur ou validateur, vous pourrez effectuer des annotations et des analyses fonctionnelles sur le génome bactérien.

- Sinon, si vous choisissez le rôle de validateur, vous avez la permission de valider ou de rejeter les annotations. Vous êtes entièrement libre de choisir le rôle que vous voulez.

- Il y a également la page d'administration pour l'administrateur GenAnnot, où l'administrateur peut consulter des informations sur les utilisateurs enregistrés et supprimer les utilisateurs indésirables. Néanmoins, le rôle d'administrateur ne peut pas être choisi dans le formulaire d'inscription.

/!\ Information importante : Si vous avez déjà un compte, vous ne pouvez pas vous réinscrire sur le site. Seuls les utilisateurs existants du site peuvent consulter et rechercher des informations dans la base de données GenAnnot.
